Welcome to the official repository for **Kamilimu Cohort 10 Data Science Classes**! This repository hosts all course materials, notebooks, and resources for the 2026 cohort.

## 🚀 Getting Started

### Prerequisites

- Computer with internet connection
- Google account (for Google Colab)
- Basic computer literacy

### Required Tools

- **Python 3.8+** - Programming language
- **Jupyter Notebook / Google Colab** - Interactive development environment
- **Git** - Version control
- **VS Code** (recommended) - Code editor

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/WanjiruCate/Kamilimu-2026-DS.git
   cd Kamilimu-2026-DS
   ```

2. Install required Python libraries:
   ```bash
   pip install jupyter pandas numpy matplotlib seaborn scikit-learn
   ```

3. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

Alternatively, use **Google Colab** (no installation required):
- Visit [colab.research.google.com](https://colab.research.google.com)
- Upload notebooks or connect to GitHub
